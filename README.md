# Juegos Gratis Now

Web app responsive para descubrir juegos gratis legales (giveaways + free-to-play), optimizada para iPhone.

## Funciones

- Búsqueda por nombre
- Filtro por plataforma
- Lista de giveaways activos
- Lista de juegos free-to-play
- PWA básica (se puede agregar a pantalla de inicio)

## Ejecutar local

```bash
cd juegos-gratis-web
python3 -m http.server 8080
```

Abrir en navegador:
- http://localhost:8080 (Mac)
- http://IP-DE-TU-MAC:8080 (iPhone en misma red)
